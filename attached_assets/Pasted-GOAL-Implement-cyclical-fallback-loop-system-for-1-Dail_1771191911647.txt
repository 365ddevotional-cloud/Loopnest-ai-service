========================================
PART 1 — DAILY DEVOTIONAL LOOP
========================================

Modify daily devotional GET route.

LOGIC:

1. Get today's date (YYYY-MM-DD).
2. Try to fetch devotional WHERE date = today.
3. If exists → return it normally.
4. If NOT exists:
   a. Get earliest devotional date in DB.
   b. Get total devotional count.
   c. Calculate days difference between today and first devotional date.
   d. Compute index = daysDifference % totalCount.
   e. Fetch devotional ordered by date ASC OFFSET index LIMIT 1.
   f. Return that devotional.

IMPORTANT:
- This must NOT modify stored dates.
- This is selection logic only.
- Works forever.

PART 2 — SUNDAY SCHOOL LOOP
========================================

Modify Sunday School GET route.

LOGIC:

1. Get current Sunday date.
2. Try to fetch lesson WHERE sundayDate = currentSunday.
3. If exists → return it.
4. If NOT exists:
   a. Get earliest sundayDate.
   b. Get total Sunday School lesson count.
   c. Calculate weeks difference from first sundayDate.
   d. index = weeksDifference % totalCount.
   e. Fetch lesson ORDER BY sundayDate ASC OFFSET index LIMIT 1.
   f. Return that lesson.
